# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 5.x     | ✅        |
| < 5.0   | ❌        |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

To report a security issue, use [GitHub's private vulnerability reporting](../../security/advisories/new).

You can expect:
- A response within **48 hours**
- A fix or mitigation within **90 days**, depending on severity
- Credit in the release notes if you'd like

Thank you for helping keep AEGIS safe.
